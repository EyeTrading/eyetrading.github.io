## 閃電下單+多週期K線

!!! example "閃電K線"

    1. <span style="color:red;font-weight:bold;">一鍵置中</span>: 右鍵點擊可以快速將五檔價格拉回中間位置.
    2. <span style="color:red;font-weight:bold;">多時間週期K線</span>: 直接將個股的1分K、5分K、日K 整合在閃電下電介面上，並能即時更新.
    3. <span style="color:red;font-weight:bold;">閃電下單整合均線</span>: 閃電下單欄位整合1分K的均線位置，同時也提示日K均線價.
    4. <span style="color:red;font-weight:bold;">當日預估量</span>: 計算個股當日預估量及昨日成交量.
