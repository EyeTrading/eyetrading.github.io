## Shioaji API 設定

!!! info "欄位說明"

    ![shioaji api settings](img/settings_screen.png){width="400"}

    - API Key: 請填入 Shioaji API Key.
    - Secret key: 請填入 Shioaji Scret Key.
    - 到期日: 此API Key 的到期日.
    - 憑證路徑: 選擇憑證檔案後會代入檔案的路徑.
    - 憑證ID: 憑證的ID.
    - 憑證密碼: 憑證使用的密碼.
    - 不儲存離開: 直接關閉視窗.
    - 儲存並離開: 設定完成之後，下次就不用再設定相關資訊.
