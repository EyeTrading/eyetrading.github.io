    富邦Neo API 操作記錄--日內行情

    富邦Neo API

    Neo API為富邦證券最近推出新的跨平台API，其提供三種程式語言:Python、Nodejs、C#，詳細的介紹可以參考富邦網站，這裡將記錄在操作api的相關過程。

    富邦Neo API 日內行情

    Web API

    # 首先需登入 API
from fubon_neo.sdk import FubonSDK

sdk = FubonSDK()
res = sdk.login("登入ID", "登入密碼",  "憑證路徑",  "憑證密碼")

# 登入成功後，取得 account資料。
if res.is_success:
    account = res.data

===============
# 日內行情 初始設定

sdk.init_realtime()     # 建立行情連線
rest_stock = sdk.marketdata.rest_client.stock          # 這個會一直用到

    Tickers

    股票代碼與名稱

    • type:
      • EQUITY: 股票(含ETF).
      • INDEX: 指數.
      • WARRANT: 權證.
      • ODDLOT: 盤中零股.
    • exchange:
      • TWSE: 臺灣證券交易所
      • TPEx: 證券櫃檯買賣中心
    • market:
      • TSE: 上市
      • OTC: 上櫃
      • ESB: 興櫃一般板
      • TIB: 臺灣創新板
      • PSB: 興櫃戰略新板
    • industry:
      • 請參考連結
    # Tickers
res = rest_stock.intraday.tickers(type='EQUITY')
print(res)

    • symbol: 股票代碼

    • timeframe: 1, 5, 10, 15, 30, 60. 相對k線的 分 週期

    未指定 timeframe 則為當日 1分K資料.
