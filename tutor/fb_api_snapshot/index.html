    富邦Neo API 操作記錄--行情快照

    富邦Neo API

    Neo API為富邦證券最近推出新的跨平台API，其提供三種程式語言:Python、Nodejs、C#，詳細的介紹可以參考富邦網站，這裡將記錄在操作api的相關過程。

    富邦Neo API 行情快照

    Snapshot

    # 首先需登入 API
from fubon_neo.sdk import FubonSDK

sdk = FubonSDK()
res = sdk.login("登入ID", "登入密碼",  "憑證路徑",  "憑證密碼")

# 登入成功後，取得 account資料。
if res.is_success:
    account = res.data

sdk.init_realtime()     # 建立行情連線
rest_stock = sdk.marketdata.rest_client.stock          # 這個會一直用到
